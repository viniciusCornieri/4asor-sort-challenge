Comandos de decisão/seleção: (if, else, switch-case, ternário :?, try/catch )
Comandos de repetição: (for, while, do-while, repeat, loop, jump)
Funções recursivas
API Pronta
Collections e Arrays
Internet/ChatGPT
Libs que utilizem as restrições anteriores
